## How to Play

### Option 1: Use the Run Script (Recommended)
Simply double-click `run.bat` or run it from command line:
```bash
.\run.bat
```

### Option 2: Manual Command
```bash
cd bin
java Main
```

## Game Controls

Once the game starts:
1. **Main Menu**: Choose "New Game" to start
2. **Enter your hero's name** when prompted
3. **Adventurer's Camp**: Your hub for all activities
   - Explore Dungeon - Fight enemies
   - Character Status - View stats
   - Inventory & Equipment - Manage items
   - Shop - Buy equipment
   - Rest - Restore HP
   - Quit Game - Exit

## Tips for Your First Adventure

1. **Check your starting inventory** - You have a Wooden Sword and Leather Armor
2. **Equip your gear** - Go to Inventory & Equipment and equip both items
3. **Save some gold** for potions at the shop
4. **Rest after battles** to restore HP
5. **Level up** by defeating enemies to get stronger

## Combat Tips

- **Attack** - Main damage dealer
- **Use Item** - Heal with potions during tough fights
- **Flee** - 50% chance to escape if you're in trouble
